Why Linux?

This question always does raise some interesting debates. I can only talk about my experiences and opinions as there is no single correct answer. My main reason is flexability. Flexability on what is installed, how it is installed, and how it behaves. To date, I have yet to find another Operating System which has such attributes.

One of the key concepts in Linux is that everything is a file. In actual fact when you drill down to the basics this is true for every operating system however it is often not portrayed as such. In Linux, an application is a file, the configuration is a stored in a file, all the libraries are files, and even the service launcher, GUIs, locking mechanisms, and logs are all files. For this reason, it is easy to support, configure, and troubleshoot virtually any Linux application, even one which one has not come accross before. There is far less rooting around looking for where the configuration is stored, what format its in, and how/if it can be modified.
